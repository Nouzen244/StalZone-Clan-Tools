STALCRAFT Clan Bot

Инструкция по установке v2.0

Прогресс:
0/6

⚡ Быстрый старт

Для запуска бота тебе понадобится выполнить 6 шагов. Примерное время: 10-15 минут.

🐍 Python 3.10+ 🤖 Discord аккаунт 💻 Терминал/CMD
1

Создай Discord бота

  1. 1. Перейди на Discord Developer Portal
  2. 2. Нажми "New Application" → введи имя (например: STALCRAFT Clan Bot)
  3. 3. Перейди в раздел "Bot" слева
  4. 4. Нажми "Reset Token" и скопируй токен (сохрани его!)

⚠️ Важно! Токен — это пароль бота. Никому его не показывай и не публикуй на GitHub!

Открыть Developer Portal
2

Включи Privileged Intents

В разделе "Bot" прокрути вниз до "Privileged Gateway Intents" и включи:

👥
SERVER MEMBERS
Обязательно!
💬
MESSAGE CONTENT
Обязательно!
🎭
PRESENCE
Опционально

🚨 Без этих разрешений бот не увидит участников сервера и голосовые каналы!

3

Пригласи бота на сервер

  1. 1. Перейди в раздел "OAuth2" → "URL Generator"
  2. 2. В SCOPES выбери: bot и applications.commands
  3. 3. В BOT PERMISSIONS выбери:
Administrator или отдельно: View Channels Send Messages Embed Links Read Message History Connect View Voice Channel

4. Скопируй сгенерированный URL и открой в браузере:

https://discord.com/oauth2/authorize?client_id=YOUR_ID&permissions=...
4

Установи Python и зависимости

Проверь, установлен ли Python:

python --version

Нужна версия 3.10 или выше. Скачать Python

Создай папку проекта и перейди в неё:

mkdir clanbot && cd clanbot

Установи зависимости:

pip install discord.py openpyxl aiosqlite pytz python-dateutil aiohttp
5

Настрой config.json

Открой файл config.json и вставь свой токен:

{
    "TOKEN": "ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН",
    "PREFIX": "!",
    "LOG_LEVEL": "INFO",
    "DB_PATH": "data/bot_database.db",
    "EXCEL_PATH": "data/clan_attendance.xlsx",
    "TIMEZONE": "Europe/Moscow"
}

🔑 Где взять токен?

Developer Portal → твоё приложение → Bot → Reset Token

🌍 Временные зоны

Europe/Moscow, Europe/Kiev, Asia/Almaty и т.д.

6

Запусти бота! 🚀

python main.py

Если всё правильно, увидишь:

2024-01-15 20:00:00 | INFO     | ClanBot | ==================================================
2024-01-15 20:00:00 | INFO     | ClanBot | 🎮 Бот запущен: STALCRAFT Clan Bot#1234 (123456789)
2024-01-15 20:00:00 | INFO     | ClanBot | 📡 Подключён к 1 серверам
2024-01-15 20:00:00 | INFO     | ClanBot | ==================================================
2024-01-15 20:00:00 | INFO     | ClanBot | ✅ Синхронизировано 0 slash-команд

🎉 Поздравляю! Бот запущен!

Теперь напиши на сервере:

!setup — мастер настройки

⚙️ После запуска — настрой на сервере

1. Установи голосовой канал:

!setvc #рейд-войс

2. Настрой роли:

!setrole leader @Глава !setrole officer @Офицер !setrole member @Клан

3. Добавь расписание:

!schedule add 20:00-21:30 Вечерний сбор

4. Проверь статус:

!status

🔧 Частые проблемы

❌ "Неверный токен" при запуске

1. Убедись, что скопировал токен полностью (без пробелов)

2. Сгенерируй новый токен через "Reset Token"

3. Проверь кавычки в JSON — должны быть обычные ""

❌ Бот не видит участников/каналы

Включи Privileged Intents (Шаг 2):

Developer Portal → Bot → SERVER MEMBERS INTENT ✅

❌ ModuleNotFoundError: No module named 'discord'

Установи зависимости:

pip install discord.py
❌ Бот не отвечает на команды

1. Проверь префикс (по умолчанию !)

2. Включи MESSAGE CONTENT INTENT

3. У бота есть права видеть канал?
